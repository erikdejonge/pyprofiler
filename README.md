# pyprofiler
Profiler utility for python

###profile some code
```python
profiler = start_profile()
meth1()
meth2()
meth3()
meth4()
end_profile(profiler)
```

###graphical representation, needs wxpython and runsnakerun
```bash
brew install wxPython
pip install runsnakerun
```
